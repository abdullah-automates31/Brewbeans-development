# ☕ Brew Beans

A modern, responsive coffee shop website built with HTML, CSS, JavaScript, and Bootstrap. Brew Beans provides customers with a premium online experience, allowing them to explore the menu, place orders, track their orders, and learn more about the coffee shop.

## 🚀 Features

- Responsive design for all devices
- Modern and elegant UI
- Interactive navigation
- Coffee menu display
- Online ordering interface
- Order tracking page
- Staff page
- Smooth animations and effects
- Fast loading performance
- SEO-friendly structure

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## 💻 Installation

1. Clone the repository

```bash
git clone https://github.com/abdullah-automates31/Brewbeans-development.git
```

2. Open the project folder

```bash
cd Brew-Beans
```

3. Open `index.html` in your browser.
